# Data Layers

This directory documents the logical data layers used in this project.
Physical data storage locations may vary by environment and are not tracked directly in this repository.

## Planned Layers

- **Raw**: Immutable data ingested directly from source systems
- **Processed**: Cleaned and standardized datasets
- **Analytics** (future): Business-ready tables for analysis

Folder structures and naming conventions will evolve as the project
matures.
